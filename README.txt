Для сборки - make all.
========
Набор тестовых программ для лабораторной № 1 по ОСП. 

01. lab1aaaN3248
------------ 
Программа для вызова из плагина функции проверки файла. Дополнительно выводит информацию о плагине. Выводит список файлов которые прошли проверки (плагина).
Пример запуска:
./lab1aaaN3248 -P ~/lab1 --ipv4-addr-bin 192.168.8.1 ~/lab1/lab1aaaN3248/test/

02. libaspN3248.so
------------    
Плагин для лабораторной № 1. Позволяет искать ipv4 адрес в бинарной форме в файлах. Функция plugin_process_file() возвращает 0, если в файле было найдено двоичное представление данного ipv4 адреса (как в little-endian, так и в big-endian) заданного ip адреса, и 1 в противном случае.
Поддерживаемые опции:
 --ipv4-addr-bin D.D.D.D
 Пример: --ipv4-addr-bin 192.168.8.1
